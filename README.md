# 🍔 Food Delivery Time Prediction

## 📌 Problem Statement

In the fast-growing food delivery industry, estimating accurate delivery time is crucial for customer satisfaction.
This project aims to predict the delivery time of food orders based on multiple real-world factors affecting delivery performance.

---

## 🚀 Solution

Developed a Machine Learning model to predict food delivery time using historical order data.
The solution includes data preprocessing, feature engineering, model training, evaluation, and deployment through an interactive Streamlit web application.

---

## 📊 Dataset

* Source: Kaggle (Food Delivery Dataset)
* Features include:

  * Delivery distance
  * Weather conditions
  * Traffic level
  * Order type etc.
  
## ⚙️ Tech Stack

* Python
* NumPy, Pandas
* Matplotlib, Seaborn
* Scikit-learn
* Streamlit

---

## 🔍 Workflow

1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building (Regression)
5. Model Evaluation
6. Deployment using Streamlit

---

## 🤖 Model Used

* A simple linear regression 

---

## 📈 Results

* Achieved strong prediction performance on test data
* Evaluation Metrics:

  * R² Score: 0.85+
  * RMSE: Low error indicating good prediction accuracy

---

## 🌐 Web App (Streamlit)

An interactive web application built using Streamlit allows users to input delivery-related features and get predicted delivery time instantly.


## 💡 Key Learnings

* Handling real-world datasets
* Feature engineering techniques
* Model evaluation and tuning
* Building and deploying ML apps using Streamlit
